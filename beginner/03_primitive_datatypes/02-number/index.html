Introduction to Number Data Types in JavaScript

In JavaScript, numerical values are represented using the Number data type. This type can store both integers and floating-point numbers. All numbers in JavaScript use the 64-bit floating-point format defined by the IEEE 754 standard.

1. Basic Examples

Examples of number values in JavaScript:
let a = 10;
let b = 3.14;
let c = -25;

2. Special Number Values

JavaScript also includes special numeric values:
  • Infinity: Represents positive infinity. Example: 1 / 0
  • -Infinity: Represents negative infinity. Example: -1 / 0
  • NaN (Not a Number): Indicates an invalid numerical result. Example: "abc" * 2

3. BigInt

BigInt is a separate numeric type used for arbitrarily large integers beyond the safe integer limit.

Example: let big = 123456789012345678901234n;
Note: You cannot mix BigInt and Number types directly in operations.

4. Useful Methods

Common methods for working with numbers:
  • Number.isNaN(value) – Checks if a value is NaN
  • Number.isFinite(value) – Checks if a value is a finite number
  • parseInt("123abc") – Converts a string to an integer: 123
  • parseFloat("3.14xyz") – Converts a string to a float: 3.14

5. Number Properties

JavaScript provides several built-in constants related to numbers:
  • Number.MAX_VALUE – Largest positive number
  • Number.MIN_VALUE – Smallest positive number (closest to 0)
  • Number.MAX_SAFE_INTEGER – 2^53 - 1
  • Number.MIN_SAFE_INTEGER – -(2^53 - 1)

6. Type Checking

typeof 42 → "number"
typeof 42n → "bigint"
