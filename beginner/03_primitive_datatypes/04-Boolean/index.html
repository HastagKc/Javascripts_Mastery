Introduction to Boolean in JavaScript

1. What is a Boolean?

The Boolean type represents a logical entity and can have only two values: true or false.

2. Boolean Literals

You can directly assign boolean values:
let isAvailable = true;
let isLoggedIn = false;

3. Boolean Constructor

You can convert values to Boolean using the Boolean() function:
Boolean(1) → true
Boolean(0) → false
Boolean("Hello") → true
Boolean("") → false

4. Falsy Values in JavaScript

The following values are considered falsy (they convert to false when used in a boolean context):

  • false
  • 0 and -0
  • "" (empty string)
  • null
  • undefined
  • NaN

5. Truthy Values

Any value that is not falsy is considered truthy. For example:

  • true
  • {} (empty object)
  • [] (empty array)
  • "hello"
  • 42

6. Logical Operators

  • ! – Logical NOT: !true → false
  • && – Logical AND: true && false → false
  • || – Logical OR: false || true → true

7. typeof Operator

You can check the type using typeof:
typeof true → "boolean"
