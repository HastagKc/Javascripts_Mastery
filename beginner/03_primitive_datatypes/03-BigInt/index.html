Introduction to Number Type in JavaScript

In JavaScript, numerical values are represented using the Number data type. This type can hold both integers and floating-point numbers. JavaScript numbers are based on the 64-bit floating-point format (IEEE 754).

1. Basic Examples

Examples of numbers in JS:
let a = 10;
let b = 3.14;
let c = -25;

2. Special Number Values

JavaScript provides special numeric values:
  • Infinity: Represents positive infinity. Example: 1 / 0
  • -Infinity: Represents negative infinity. Example: -1 / 0
  • NaN (Not a Number): Returned from invalid math. Example: "abc" * 2

3. NaN Example

console.log(isNaN("abc")); → true
console.log(Number("abc")); → NaN

4. Useful Number Methods & Functions

  • Number() – Converts a value to a number.
  • parseInt() – Parses a string into an integer.
  • parseFloat() – Parses a string into a floating-point number.
  • toFixed() – Rounds and formats a number to fixed decimals.
  • toString(base) – Converts number to a string in the specified base.
    Example: (255).toString(2) → "11111111"
    Example: (255).toString(16) → "ff"
  • isNaN() – Checks if value is NaN (after type coercion).
  • Number.isNaN() – Checks if value is exactly NaN.
  • Number.isFinite() – Checks if value is a finite number.
  • Math.round() – Rounds to nearest integer.
  • Math.floor() – Rounds down.
  • Math.ceil() – Rounds up.
  • Math.trunc() – Removes decimal part.
  • Number.MAX_VALUE – Largest positive number in JS.
  • Number.MIN_VALUE – Smallest positive number in JS.
  • Number.MAX_SAFE_INTEGER – Largest safe integer: 2^53 - 1
  • Number.MIN_SAFE_INTEGER – Smallest safe integer: -(2^53 - 1)

5. Type Checking

typeof 42 → "number"
typeof NaN → "number"
typeof 42n → "bigint"

6. BigInt in JavaScript

BigInt is a special numeric type in JavaScript used to represent integers larger than Number.MAX_SAFE_INTEGER (i.e., values beyond 2^53 - 1).

You create a BigInt by appending n to the end of an integer:
let big = 1234567890123456789012345678901234567890n;
BigInt values cannot be mixed with regular Number types in arithmetic operations.

7. Useful BigInt Methods & Properties

  • BigInt() – Converts a number or string to BigInt.
    BigInt("9007199254740995") → 9007199254740995n
  • typeof – Used to check if a value is BigInt.
    typeof 10n → "bigint"
  • Supports arithmetic operators: + - * / % **
    Example: 10000000000000000000n + 1n → 10000000000000000001n
  • Cannot be used with Math methods like Math.sqrt(), Math.random().
